Секция Настройки Яндекс.Вебмастер предназначена для получения кода и хранения авторизационного токена для данного сервиса.

Полная документация

При нажатии на ссылку "Получить код" будет выполнено перенаправление на страницу подтверждения запроса. Нажмите "Подтвердить" и вам будет выдан код подтверждения, который необходимо будет ввести в поле "Введите код подтверждения" и нажать кнопку "Сохранить". После сохранения, в секции будет отображаться ваш текущий токен в соответствующем поле.

Секция SEO настройки доменов предназначена для хранения и изменения настроек SEO для каждого домена системы.

В поле Префикс для TITLE можно задать префикс для заголовков окна. В поле TITLE (по умолчанию) задается заголовков окна для страниц, у которых не заполнено поле "title".

С версии 2.8.5.1 появилась возможность вставлять значение поля H1 не только в конце, но в любом необходимом месте строки. Для этого следует использовать "макрос" %title_string%. Поместите его туда, где необходимо вставить значение поля H1 текущей страницы. например, конструкция "UMI.CMS :: %title_string% :: Современный интернет-магазин" на конкретной странице даст Title страницы "UMI.CMS :: DVD-диски :: Современный интернет-магазин".

В нижних двух полях можно указать мета-теги, которые будут подставлены по умолчанию, если соответствующие поля не заполнены у какой-либо страницы или публикации.
